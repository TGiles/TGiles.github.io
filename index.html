About Me

I am a recent graduate from Christopher Newport University with a B.S. and M.S. (Thesis) in Computer Science. I enjoy solving problems using proper software engineering principles and also enjoy implementing these principles in an organization. Personally, I am a believer in the Agile methodology and is something I like to bring to organizations. I also enjoy contributing to open source when possible.

For whatever reason, I decided to write a thesis in a field that I had zero experience in. My thesis has to be the most challenging thing I have done to date. Because of this thesis, I now understand robotics better and the tools/frameworks that roboticists use to create applications.

I have worked as a software engineer, graduate lab assistant, tutor, and tech camp instructor across seven years. I enjoy learning exciting and relevant technologies, as well as automation processes. Thanks to my Master's Thesis, I also learned that I enjoy robotics and AI, at least from a software perspective. Additionally, I had to learn ROS (Robot Operating System) in order to fully utilize this sytem.

I have been in a few bands over the years, playing drums or bass. The most recent band I was in is DOWNHAUL. They are an excellent group of guys that make killer indie rock/emo music, if that is your thing. On the other side of creativity, I enjoy photography and creating interesting shots. I currently use a Nikon D3300 with 35mm, 18-55mm, and 55-300mm lenses.

One day I'd like to own and run a record label. Imagine how cool that would be, helping other peoples' dreams come true!

Undergraduate Capstone

The gist of this project was to create a academic gatekeeping application using physics papers. In detail, I wanted to create a data visualization project using D3js as the main tool. I ended up using JavaScript, Java, PHP, MySQL, Bootstrap3, and D3js to create the entire project. The end result is a force-directed graph that shows citation relationships between papers. This allows a user to visually see which papers are important in a citation network.
